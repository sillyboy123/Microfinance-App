Testing Instructions

The tests above check your application's connectivity to the database. If you're experiencing issues, follow these steps:

  1. Check the connection string in appsettings.json
  2. Verify SQL Server or LocalDB is running and accessible
  3. Run SetupLocalDB.bat if you want to use SQL Server LocalDB
  4. Run SetupSQLiteBackup.bat if you want to use SQLite instead

The application is designed to fallback to SQLite if SQL Server connectivity fails, ensuring data persistence in various environments.

About Database Connections

The Microfinance App supports multiple database providers:

  • SQL Server - Full-featured relational database
  • LocalDB - Lightweight SQL Server for development
  • SQLite - File-based database for fallback scenarios
